# 💬 Simple Chat App

🚀 A clean and lightweight Android Chat Application
📱 Built using **Java + Android Studio**

---

## ✨ Features

* 🔐 Login Screen
* 📝 Registration Screen
* 👥 User List
* 💬 Real-time Chat (Demo)
* 🎨 Clean & Modern UI

## 🧠 How It Works

1. User logs in or registers
2. Selects a user from list
3. Opens chat screen
4. Sends and receives messages
